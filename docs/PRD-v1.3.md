# ⚡ 《Grid Master: Cyber Network》v1.3 美術規格（正式視覺第一輪）

> 2026-07-12 定案：整體風格＝**極簡扁平＋霓虹點綴**（延續深藍黑底＋青紫色系，
> 全程式產出 SVG／PixiJS，零外部素材依賴）；卡牌＝**圖騰＋幾何紋**（不靠插圖）。
> 本輪範圍：A1 圖示系統 → A2 地圖換皮 → A3 盤面動畫。介面精修（字型／logo／
> 按鈕質感）與 Rive 留待下輪。

## 0. 風格基準（Style Tokens）

- **色票**：沿用現有 CSS 變數——底 `#0b0e17`／面板 `#121627`／浮起 `#1a2038`、
  主青 `#37e6d4`、輔紫 `#9d6bff`、警示紅 `#ff5470`、正向綠 `#4ade80`。
  新增資源專色：水力青藍、火力橘、廢料綠、算力粉紫（自圖示系統定案）。
- **形狀語言**：直角＋45° 切角（chamfer）、細描邊（1.5–2px）、發光只用在
  「當前可互動」與「事件瞬間」，常態畫面保持低亮度。
- **原則**：所有視覺元件皆為原創幾何構成（PRD §7 聲明——不引用原作素材）。

## A1 圖示系統（先做——其他兩項的地基）

**目標**：自製 SVG 圖示全面取代 emoji，跨平台渲染一致。

- 圖示集：
  - 4 資源：💧水力、🔥火力、♻️廢料、🧠算力 → 幾何圖騰（水滴電路／火焰晶格／
    回收迴圈／神經節點），各配專色。
  - 7 設施類型：hydro/thermal/waste/quantum/hybrid（雙色分割）/self（葉脈電路）/
    fusion（漩渦）。
  - 通用：⚡能量（貨幣＋供電）、🏙節點數、💰餘額。
- **交付形式**：`src/components/GameIcon.vue`（inline SVG、`currentColor`＋專色
  雙模式、`size` prop）；`RESOURCE_META`／`TYPE_META` 增加 `color` 欄位。
- **替換範圍**：面板類 UI 全換（市場梯、儲存凹槽、採購列、玩家卡、卡牌、
  規則彈窗、詳情彈窗）。**純文字區（事件日誌、聊天、tooltip）暫留 emoji**，
  避免文字流變成元件流的重構，下輪再議。
- **卡牌視覺**（依 2026-07-12 定案的圖騰＋幾何紋）：
  - 卡面＝類型圖騰（大）＋類型色帶＋依卡號 seed 的幾何裝飾紋（等高線／電路走線，
    卡號越大紋越密）——純 SVG 程式生成，42 張各不相同但同語言。
  - `PlantCard.vue` 版型隨之調整（圖騰區＋數值區）。

## A2 地圖換皮

**目標**：從「圓圈＋直線」升級為數據中心網路的視覺敘事。

- **節點**：圓 → 切角方形晶片章（region 色細框＋內部針腳紋）；佔據者以
  玩家色「插槽燈」呈現（取代現在的小圓點）；可建目標維持青色高亮＋外框呼吸。
- **線路**：加上淡淡的「走線」質感（雙細線或中央虛線），過路費標籤配半透明
  底板（放大時更好讀）。
- **背景**：PixiJS 程式生成的電路板紋理層（低對比格點＋稀疏走線，隨機 seed
  固定），跟著 viewport 縮放平移。
- **叢集**：region 色調的角落標記或底色暈染，未啟用叢集維持反灰。
- 兩種佈局（地理／網格）共用同一套視覺。

## A3 盤面動畫（事件驅動，吃 `game_events`）

- `city_built`：節點插槽燈點亮＋一圈擴散波。
- `plant_bought`（競標成交）：市場對應卡短暫高亮（DOM/CSS 即可）。
- `powered`：官僚結算時，供電玩家的網路線路跑一次「電流」粒子（沿邊流動的
  光點），未供電節點不亮——把「供電數 ≤ 城市數」視覺化。
- `step_changed`／`step3_revealed`：全版短暫掃描線過場（Pixi 覆蓋層）。
- **效能底線**：動畫全部一次性（無常駐粒子），手機 60fps；`prefers-reduced-motion`
  時全部跳過。
- 事件已有序送達（M2 設計），前端僅消費 `room.lastEvents`，不動協議。
