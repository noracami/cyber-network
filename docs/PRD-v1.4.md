# ⚡ 《Grid Master: Cyber Network》v1.4 需求規格（NPC 與單人遊玩）

> 2026-07-12 定案並實作。目標：一位玩家配 NPC 就能開局遊玩／測試，
> 不再需要湊人或開多個無痕視窗。

## R1 NPC（不求勝）

- **大腦**（`lib/grid_master/npc.ex`）：對引擎狀態產生「偏好排序的候選動作
  清單」，Room 依序嘗試執行；每個階段的清單都以必定合法的動作收尾
  （棄權／空買／結束擴建／空供電）——與 `random_drive_test` 相同的守恆模式，
  保證牌局永不卡在 NPC 手上。
- **策略**（樸素貪婪，刻意不求勝）：提名最便宜的卡（手上 ≥2 座傾向觀望；
  第一回合必買規則自然生效）、出價上限「卡號＋3」、採購買齊一輪供電所需
  燃料（混合廠偏水力）、擴建最便宜可達城且保留 $15、供電全開（資源不足
  逐次拿掉需求最大者）。
- **接入**：NPC 住在 Room GenServer 旁——不開 WebSocket。每次狀態變更後
  檢查「輪到 NPC？」，`Process.send_after` 延遲 0.9–1.8 秒（擬人）觸發
  `:npc_tick` 出手，走與真人相同的 `Engine.apply_action` 管線；事件流、
  動畫、日誌全部沿用，前端無感。
- **座位**：大廳「＋ NPC／－ NPC」鈕（登入使用者限定，與入座同門檻）；
  NPC 名字取電學單位（🤖 伏特／安培／歐姆／瓦特／焦耳／法拉）、恆為
  已準備、恆在線、跨局保留座位、不受離線逾時清理影響。

## R2 開始／結束權力下放

- **開始**：規則不變（任一入座玩家在全員準備後可開局）——1 真人＋≥1 NPC
  即滿足人數下限；純 NPC 局不可能發生（開局者必須入座）。
- **結束**（修訂 PRD §3.3 的 Admin 專屬 ABORT）：**本局入座玩家**也可
  結束遊戲回大廳（頂欄「結束遊戲」鈕）；Admin 保留原「掀桌」。旁觀者不可。
  單人配 NPC 遊玩時得以自行收局。

## 順帶修正

- 聊天時間戳改用瀏覽器本地時區顯示（原為 UTC 切字串，+8 時區慢八小時）。

## 驗證

- 引擎層：2／4 NPC 全 NPC 對局以候選策略驅動至第 3 回合以上不卡死。
- Room 層：1 真人＋1 NPC 開局、NPC 自動出手推進至第 2 回合、
  入座玩家可結束／旁觀者不可（`npc_test.exs`，共 101 測試綠）。
- 瀏覽器 E2E：註冊 → 入座 → ＋NPC → 開局 → NPC 自主提名 →
  玩家「結束遊戲」回大廳，全程無錯誤。
